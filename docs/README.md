- **CLIP_SETUP.md** — CLIP / re-ranking service used by detection pipelines.
- **MAPPING_GUIDE.md** — Autonomous SLAM mapping workflow on TIAGo.
- **MAP_INTEGRATION_GUIDE.md** — Importing external maps into the navigation stack.
- **SAFETY_FEATURES.md** — Safe-mode behavior and shutdown procedures.
- **GRASPING_IMPROVEMENTS.md** — Grasp pipeline technical notes and calibration tips.
- **IMPROVEMENTS_SUMMARY.md** — High-level release highlights with links to detailed docs.

- **TECHNICAL_REPORT.md** — Full system architecture and methods.
